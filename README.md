# 射击游戏

一个完整的双人对战射击游戏，使用 HTML5 Canvas 和 JavaScript 开发。

## 🎮 游戏说明

### 游戏规则
- **玩家血量**：10滴血
- **对手血量**：10滴血
- **子弹数量**：可设置（默认8发）
- **伤害值**：每次击中造成2点伤害
- **胜利条件**：先击败对手的一方获胜

### 操作方式
- **移动**：使用 **WASD** 键控制角色移动
  - **W** - 向上移动
  - **A** - 向左移动
  - **S** - 向下移动
  - **D** - 向右移动
- **瞄准**：移动鼠标到画布上，瞄准对手（瞄准线会显示方向）
- **射击**：按 **空格键** 射击
- **重新装弹**：点击"重新装弹"按钮补充子弹
- **重新开始**：点击"重新开始"按钮重置游戏

### 游戏设置
- **子弹数量**：可设置 1-50 发
- **子弹速度**：可设置 3-20（数值越大速度越快）

### 功能特性
- ✅ 实时瞄准系统（鼠标控制）
- ✅ 子弹物理系统（轨迹计算）
- ✅ 碰撞检测（精确命中判定）
- ✅ AI对手（自动移动和射击）
- ✅ 血量系统（可视化血量条）
- ✅ 子弹计数（实时显示剩余子弹）
- ✅ 游戏结束判定（胜负判定）
- ✅ 可自定义设置（子弹数量、速度）

## 🛠️ 技术实现

### 核心系统
1. **射击系统**
   - 空格键触发射击
   - 子弹方向计算（从玩家位置到瞄准点）
   - 可配置的子弹速度

2. **移动系统**
   - WASD键控制
   - 平滑移动
   - 边界检测

3. **AI系统**
   - 对手自动上下移动
   - 对手自动瞄准玩家
   - 定时射击机制（2秒间隔）

4. **碰撞检测**
   - 子弹与角色矩形碰撞
   - 精确的命中判定

5. **游戏状态管理**
   - 游戏运行/暂停/结束状态
   - 血量管理
   - 子弹计数
   - 可配置参数

### 视觉效果
- 渐变背景
- 网格背景
- 瞄准线（虚线）
- 瞄准点（红色圆圈）
- 血量条（角色上方和UI面板）
- 子弹光晕效果

## 🎯 游戏技巧

1. **移动躲避**：使用WASD键移动，躲避对手子弹
2. **瞄准技巧**：预判对手移动方向，提前瞄准
3. **子弹管理**：合理使用子弹，必要时重新装弹
4. **快速射击**：在对手移动缓慢时快速射击
5. **调整设置**：根据自己喜好调整子弹数量和速度
